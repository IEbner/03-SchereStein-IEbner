Schere Stein Papier

Kurzbeschreibung:

In Schere Stein Papier spielt ein Player gegen einen Computer. Durch das Drücken des „Spielen“ – Buttons wird willkürlich ausgewählt welches Symbol die beiden spielen.

Ausgewählt wird jeweils zwischen Schere, Stein und Papier. Je nachdem, was gewählt wurde, steht ein Gewinner fest.

Bei gleichen Symbolen ist die Runde unentschieden, wo hingegen bei ungleichen Symbolen kontrolliert werden muss, welches stärker ist. Schere schneidet Papier, wird aber von Stein erschlagen. Wiederum wird Stein von Papier umhüllt.

Entwicklungsplattform:

Windows 10, Visual Studio 2019, Unity 2020.3.18f1

Lessons learned

Anwendung von if Statements, randomRange Funktion, Serializefield
